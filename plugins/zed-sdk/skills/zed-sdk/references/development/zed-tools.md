## ZED Tools Overview

> Source: https://www.stereolabs.com/docs/development/zed-tools/

The ZED Tools are demonstration applications designed to help users explore and understand their ZED camera's capabilities. They offer a user-friendly interface to visualize and interact with data captured by the device.

> **Note**: These tools are not intended for production environments and do not expose the full range of features or advanced configuration options available in the corresponding ZED SDK modules.

### Available Tools

The ZED Tools suite includes eight applications available on both Windows and Linux:

**Core Visualization Tools:**

- **ZED Explorer** - View raw video feeds, adjust camera controls, record SVO files, access raw calibration parameters. Works with both stereo and monocular cameras.
- **ZED Studio** - Simultaneous viewing and management of multiple ZED cameras, providing a centralized multi-camera stream viewer.
- **ZED Depth Viewer** - Visualization of depth maps and point cloud data generated by ZED stereo cameras.

**Sensor and Configuration Tools:**

- **ZED Sensor Viewer** - Monitors IMU data and other camera sensor information. For ZED 2i users, includes a calibration interface for the magnetometer sensor.
- **ZEDfu** - 3D reconstruction and environment visualization using the ZED SDK's Spatial Mapping module.

**Maintenance and Calibration Tools:**

- **ZED Diagnostics** - Health and performance checks on ZED cameras, assisting with both software and hardware troubleshooting.
- **ZED Calibration** - Recalibration for specialized scenarios such as underwater use or shooting through glass, compensating for image distortions in non-standard conditions.
- **ZED360** - Calibrates multi-camera ZED systems and generates extrinsic parameters for the Fusion module.

## ZED Explorer

> Source: https://www.stereolabs.com/docs/development/zed-tools/zed-explorer/

ZED Explorer is a diagnostic and monitoring tool that verifies hardware and software functionality for ZED cameras. It enables visualization of raw video feeds, camera control testing, and access to calibration parameters. Supports both stereo and monocular camera configurations.

Key capabilities:
- Live video stream preview
- SVO (native ZED SDK format) recording and playback
- Local network streaming server functionality
- Real-time camera parameter adjustment
- High-resolution snapshot capture

### Running ZED Explorer

**Windows:**

```
C:\Program Files (x86)\ZED SDK\tools\ZED Explorer.exe
```

**Ubuntu/Linux:**

```bash
ZED_Explorer
```

Or navigate to: `/usr/local/zed/tools/ZED_Explorer`

### Camera Selection

Upon launch, the application auto-detects connected ZED cameras and loads the first available device. Switch between multiple cameras using the dropdown menu in the top-right area. The refresh icon reloads the camera list after device reconnection. The gear icon accesses camera and application configuration dialogs.

### Video Mode Selection

Resolution and framerate options appear in top-left dropdown menus. Available modes depend on the connected camera model.

Playback mode indicator displays: `LIVE`, SVO `PLAYBACK`, or `STREAMING (NETWORK)`.

### Stream Status Monitoring

Status information displayed below the video feed:

| Field | Description |
|-------|-------------|
| Camera Model | Device identification |
| Resolution | Current output dimensions |
| Framerate | Frames per second |
| Frame Drop | Dropped frame count during session |
| Date/Time | Current time or SVO recording timestamp |
| Timestamp | Frame nanosecond precision indicator |

### Stream Control Functions

Bottom-left control icons:
- **Folder icon**: Open SVO files for playback
- **Network icon**: Connect to remote ZED streaming server
- **Antenna icon**: Enable/disable local streaming server
- **Record icon**: Start/stop SVO recording
- **Camera icon**: Capture current frame as snapshot

### SVO Playback Controls

During SVO file playback: start/stop/pause playback, frame-by-frame navigation via slider.

### Configuration Settings

**Application Settings:**
- Save Location: Designate SVO and screenshot storage directory
- Compression: Adjust SVO file compression levels
- Image Format: Choose PNG, JPEG, or BMP for screenshots
- Auto Naming: Enable timestamp-based file naming
- Flip Mode: Invert video orientation for upside-down mounting
- Image Enhancement: Toggle denoising and color correction

**Camera Settings:**
- Brightness, contrast, saturation, hue
- Gain and white balance
- Exposure settings with auto-adjustment option
- Sharpness and gamma correction
- Denoising filter strength
- Exposure and analog/digital gain range controls

Parameters vary by camera model; click **DEFAULT VALUES** to reset.

**Calibration Parameters:**

Raw, unrectified calibration values display for depth estimation and 3D reconstruction. The **FACTORY RESET** button restores original settings, overwriting custom calibrations applied via ZED Calibration tool.

> **Note**: Values shown differ from SDK API outputs, which display rectified parameters.

**Firmware Updates:**

The firmware tab allows checking and installing camera firmware updates. Not all ZED camera models support this feature; unavailable cameras display a disabled tab.

## ZED Studio

> Source: https://www.stereolabs.com/docs/development/zed-tools/zed-studio/

ZED Studio is a comprehensive tool for verifying ZED camera hardware and software functionality. It enables visualization of multiple camera feeds simultaneously, testing of camera controls, and performance assessment with both stereo and monocular configurations.

The tool provides the most straightforward method for recording and playing back multiple camera streams in SVO format. SVO files allow developers to replay recorded data in applications without requiring live camera access.

ZED Studio can also function as a local streaming server for connected cameras, allowing video feeds to be streamed across a local network.

### Running ZED Studio

**Windows:**

```
C:\Program Files (x86)\ZED SDK\tools\ZED Studio.exe
```

**Ubuntu:**

```bash
ZED_Studio
```

Or navigate to: `/usr/local/zed/tools/ZED_Studio`

### Live Camera Playback

Upon launching, all connected ZED cameras are automatically detected and displayed in a grid layout. The left panel lists all connected cameras where visibility can be toggled using the eye icon below each camera's serial number.

The Input tab displays connected camera information including model, serial number, resolution, frame rate, timestamp data, and current date/time.

### Configuration Settings

**General Settings:**
- Resolution: Choose from available video resolutions
- Frame Rate: Select desired playback rate
- Image View: Options include Left, Right, or Stereo (monocular cameras display only Mono view)

**Camera Settings:**
Adjustable parameters include brightness, contrast, saturation, hue, gain, white balance, exposure, sharpness, gamma, and denoising strength. Auto gain/exposure and auto white balance can be toggled. A RESET button restores default parameters.

> **Note**: Available parameters depend on camera model capabilities.

### SVO Playback

Open SVO files by clicking the folder icon in the top toolbar. Multiple SVO files can open simultaneously in separate windows, supporting both stereo and monocular formats.

Playback controls at the bottom of each stream window enable start, stop, pause, and frame navigation via slider.

The Input tab reveals SVO file properties: camera model, serial number, resolution, recorded frame rate, frame count, duration, and timestamp information.

### Network Stream Playback

Connect to ZED streaming servers by clicking the network icon and entering the server IP address and port number in the dialog window.

### SVO Recording

When receiving live or network streams, record them in SVO2 format by selecting a stream and clicking RECORD in the Record tab. Multiple streams from different cameras can be recorded simultaneously as separate SVO files.

Configuration options:
- **Output Path**: Destination folder for SVO files
- **Compression**: Lossless, H.264, H.264 Lossless, H.265, H.265 Lossless
- **Bitrate**: Target bitrate for video stream; "Auto (0)" lets the application determine optimal settings

### Local Network Streaming Server

Activate a streaming server for cameras by selecting a live stream and clicking STREAM in the Stream tab.

Configuration parameters:

| Parameter | Description |
|-----------|-------------|
| Codec | H.264 or H.265 |
| Port | Network port for streaming (use different ports per camera) |
| Bitrate (kbps) | Target bitrate for video stream |
| Adaptive Bitrate | Automatic bitrate adjustment based on network conditions |
| GOP Size (frames) | Group of Pictures size affecting compression efficiency and latency |
| Chunk Size (bytes) | Data chunk size affecting streaming performance and latency |
| Target Frame Rate | Desired playback rate or "Auto (0)" for camera frame rate |

The server IP address and port appear in the top bar of the image view window. Stop the server with the STOP button in the Stream tab.

### Command Line Options

```
$ ZED_Studio -h

Usage: ZED_Studio [options]

(-h | --help)      Shows usage information.

(-l | --list)      Lists connected cameras.

(-c | --camera)    <serial_number> [resolution] [frame_rate]
                   Opens a camera with the given serial number
                   [resolution]: (HD4K | QHDPLUS | HD2K | HD1536 | HD1200 | HD1080 | HD720 | SVGA | VGA)
                   [frame_rate]: (15 | 30 | 60 | 100)

(-s | --stream)    <ip[:port]>
                   Connects to a stream at the given IP and port.

(-r | --recording) <filepath>
                   Opens a recording at the given filepath (must be an SVO file).

Examples:
ZED_Studio --camera 12345678 HD720 60
ZED_Studio --stream 192.168.3.243:30000
ZED_Studio --recording recording.svo2
ZED_Studio -c 12345678 -s 192.168.3.243 -r recording.svo2
```

## ZED Depth Viewer

> Source: https://www.stereolabs.com/docs/development/zed-tools/zed-depth-viewer/

ZED Depth Viewer is a visualization tool for exploring depth processing capabilities of ZED cameras. It enables real-time visualization of depth maps, point clouds, and related depth data from live feeds, network streams, or recorded SVO files.

### Running the Application

**Windows:**

```
C:\Program Files (x86)\ZED SDK\tools\ZED Depth Viewer.exe
```

**Ubuntu:**

```bash
ZED_Depth_Viewer
```

Or navigate to: `/usr/local/zed/tools/ZED_Depth_Viewer`

### Camera Selection

Upon launch, the application automatically detects connected ZED cameras and opens the first available device. Select different cameras from the dropdown menu (top right). Click the arrow icon to refresh the camera list. Access camera settings via the gear icon.

### Video Mode and Source Selection

Select resolution and frame rate from dropdown menus at the top left. Three input sources supported:
- **Live Camera**: Real-time feed from connected ZED camera
- **SVO File**: Playback of recorded SVO files
- **Network Stream**: Stream from remote devices supporting ZED streaming

### Interface Sections

**Color Video Feed:** Shows left, right, or side-by-side synchronized stereo images with user-selectable viewing modes.

**Depth Feed:** Displays either depth map visualization or confidence map showing measurement reliability.

**3D Data Feed (Point Cloud):**
- Left-click + drag: Rotate view
- Mouse wheel: Zoom in/out
- Right-click + drag: Pan view

### Toggle Options

- **Depth Shading**: Apply shading effects to depth visualization
- **Lidar View**: Transform point cloud to depth-colored representation
- **Gravity Alignment**: Align point cloud using IMU data
- **Confidence Filtering**: Quick toggle between confidence levels 95 and 100

### Performance Monitoring

Bottom-left displays real-time metrics:
- Image grab frequency
- Depth processing frequency

### Configuration

**Depth Initialization** (requires camera restart):
- Depth mode selection (NEURAL_LIGHT, NEURAL, NEURAL_PLUS)
- Minimum and maximum depth range
- Depth stabilization strength

**Depth Runtime** (adjustable while streaming):
- Saturated area removal
- Confidence threshold adjustment
- Texture confidence enablement

**Data Saving Formats:**

| Type | Formats |
|------|---------|
| Image | PNG (LEFT, RIGHT, SIDE BY SIDE, DEPTH IMAGE) |
| Depth | PNG, PFM, PGM |
| Point Cloud | PLY, XYZ, PCD, VTK |

**Camera Settings:** Adjustable parameters include contrast, saturation, hue, gain, white balance, exposure, sharpness, gamma, and denoising. Auto-adjustment modes for exposure and white balance are available. Click "DEFAULT VALUES" to reset.

## ZED Sensor Viewer

> Source: https://www.stereolabs.com/docs/development/zed-tools/zed-sensor-viewer/

ZED Sensor Viewer is a tool for monitoring IMU data and sensor information from ZED cameras in real-time, including accelerometer, gyroscope, and magnetometer readings.

### Running the Application

**Windows:**

```
C:\Program Files (x86)\ZED SDK\tools\ZED Sensor Viewer.exe
```

**Ubuntu:**

```bash
ZED_Sensor_Viewer
```

Or navigate to: `/usr/local/zed/tools/ZED_Sensor_Viewer`

### Selecting a Camera

The application automatically detects all connected ZED cameras and opens the first available one. Select different cameras via a dropdown menu in the top-left corner. Click the arrows icon to refresh the list after connecting or disconnecting cameras.

### Visualization Features

**Sensor Selection Panel:**

| Sensor | Description |
|--------|-------------|
| Accelerometer | Displays acceleration along X, Y, Z axes in 3D space |
| Gyroscope | Shows rotational data around X, Y, Z axes |
| Orientation | Visualizes roll, pitch, and yaw angles |
| Barometer | Displays atmospheric pressure and altitude (ZED 2i only) |
| Magnetometer | Shows magnetic field strength along axes (ZED 2i only) |
| Temperature | Monitors internal camera temperature |

**Camera Orientation Display:** Shows 3D camera positioning relative to gravity.

**Data Plot:** Real-time graphical representation of selected sensor data over time.

### Data Recording

Click the SAVE button to export sensor data to CSV format. Available recording durations: 5, 10, 30, or 60 seconds.

Advanced settings via the gear icon:
- Data destination folder
- Which data streams to capture
- Sampling frequency

### Magnetometer Calibration

When the Magnetometer sensor is selected, a "Calibrate Magnetometer" button appears in the bottom-left corner. This feature is exclusive to ZED 2i models.

## ZEDfu

> Source: https://www.stereolabs.com/docs/development/zed-tools/zedfu/

ZEDfu is a testing tool for the Spatial Mapping module within the ZED SDK. It enables users to create 3D maps from live ZED camera feeds or pre-recorded SVO files.

### Running ZEDfu

**Windows:**

```
C:\Program Files (x86)\ZED SDK\tools\ZEDfu.exe
```

**Ubuntu/Linux:**

```bash
ZEDfu
```

Or navigate to: `/usr/local/zed/tools/ZEDfu`

### Data Source and Depth Mode Selection

Three input options:
- **Live Camera**: Connects to an active ZED stereo camera
- **Local Streaming**: Receives data from a remote ZED camera via streaming server
- **SVO File**: Processes previously recorded video files

Three depth estimation modes:
- **NEURAL LIGHT**: Faster processing with reduced quality
- **NEURAL**: Balanced performance and quality
- **NEURAL PLUS**: Maximum quality, recommended for offline processing

The tool also supports opening existing map files for visualization without regenerating them.

**SVO-Specific Features:** When using SVO files, an **OFFLINE mode** preprocesses the entire file before map generation, yielding higher-quality results at the cost of increased processing time.

### Configuration Panel

Access settings via the top-right gear icon.

**Map Generation:**
- Depth range (auto or manual)
- Positional tracking mode (GEN1, GEN2, GEN3)
- Area Memory toggle
- Output format (MESH or POINT CLOUD)
- Resolution presets (LOW, MEDIUM, HIGH)
- Map range settings
- Stability counter
- GPU memory allocation (512 MB to 8 GB)
- Filter strength levels
- Texture application option
- Maximum displayable chunks

**Live Camera Settings:**
- Gain and exposure controls with auto-adjustment capability

### Map Generation Process

**SVO Input:** Click **START** to begin processing. Preprocessing occurs automatically when enabled, followed by map generation with real-time progress updates. A completion notification displays upon success.

**Live Input:** Click **START** to capture from camera or streaming source. Move the camera to capture 3D spatial data. The **Camera Lock** button centers the view on the ZED camera position. Tracking status updates display continuously; if status switches to "SEARCHING," reorient the camera to reinitialize. Click **STOP** when complete.

> **Important**: Maps save automatically to the `ZED/Meshes` folder in the user home directory. The full path displays in the status bar upon completion.

### Map Exploration

**Navigation Controls:**
- Left mouse button: Rotate view
- Mouse wheel: Zoom in/out
- Right mouse button: Pan view

**Mesh Rendering Modes** (bottom-right toolbar):
- **Points**: Point cloud representation
- **Edges**: Edge highlighting
- **Faces**: Face highlighting
- **Shades**: Shaded rendering

**Point Cloud Mode:** Adjust point size for clarity.

## ZED Diagnostic

> Source: https://www.stereolabs.com/docs/development/zed-tools/zed-diagnostics/

ZED Diagnostic is a troubleshooting utility designed to help identify and resolve issues with ZED cameras. It provides comprehensive system and hardware status information.

### Launching the Tool

**Windows:**

```
C:\Program Files (x86)\ZED SDK\tools\ZED Diagnostic.exe
```

**Ubuntu/Linux:**

```bash
ZED_Diagnostic
```

Or navigate to: `/usr/local/zed/tools/ZED_Diagnostic`

### Running Diagnostics

After connecting your ZED camera and launching the tool, press the **Start** button to begin automated testing. Checks performed:

| Category | Details |
|----------|---------|
| ZED SDK | Version and CUDA compatibility |
| PROCESSOR | CPU model and motherboard details |
| GRAPHIC CARD | GPU identification |
| AI MODELS | Available models and optimization status |
| CAMERA TEST | Hardware tests, model, and firmware version |
| DEVICES | Connectivity verification |

The tool displays real-time progress and provides a summary upon completion, indicating whether systems function correctly. Error messages appear if issues are detected.

### Optimizing AI Models

After diagnostics complete, click the **AI MODELS** button to review model status and optimization. Select desired models and click **Optimize Selection** to download and prepare them for use. To free space or reset optimizations, click **CLEAR ALL MODELS**.

### Command Line Usage

```bash
ZED_Diagnostic --help
```

| Command | Function |
|---------|----------|
| `-d` | Auto-start diagnostics in GUI mode |
| `-c` | Command-line mode (useful without display) |
| `-aid` | Download all AI models |
| `-aio` | Optimize all AI models (downloads if needed) |
| `-aic` | Clear all AI models |
| `-ais X` | Download/optimize specific detection model by ID |
| `-nrlo` | Download/optimize NEURAL depth mode |
| `-nrlo_plus` | Download/optimize NEURAL PLUS depth mode |
| `-nrlo_light` | Download/optimize NEURAL LIGHT depth mode |
| `-nrlo_all` | Download/optimize all neural depth modes |
| `-r` | Recover MCU module with invalid serial (ZED 2/2i) |
| `-dmesg` | Retrieve kernel/driver logs for GMSL2 debugging |

### Sharing Diagnostic Reports

After failures occur, press **Save** to export results as a JSON file. Share this with Stereolabs support at support@stereolabs.com along with detailed issue descriptions.

## ZED Calibration

> Source: https://www.stereolabs.com/docs/development/zed-tools/zed-calibration/

The ZED Calibration tool enables fine-tuning of camera calibration parameters to enhance depth accuracy and 3D reconstruction. It allows adjustment of both intrinsic parameters (focal length, principal point) and extrinsic parameters (relative position and orientation of CMOS sensors).

> **Important**: The factory calibration is typically more accurate than user calibrations. Only use this tool when you have specific reasons, such as:
> - Compensating for distortions from glass placed in front of the camera
> - Operating the camera underwater
> - Observing degraded depth perception after extreme temperature exposure or physical impact

### How to Use ZED Calibration

**1. Connect the Camera**

Ensure your ZED camera is connected. The application will exit with an error message if no camera is detected.

**2. Launch the Application**

**Windows:**

```
C:\Program Files (x86)\ZED SDK\tools\ZED Calibration.exe
```

**Ubuntu/Linux:**

```bash
ZED_Calibration
```

Or navigate to: `/usr/local/zed/tools/ZED_Calibration`

**3. Follow the Tutorial (Optional)**

Click the **Tutorial** button for a 7-step guided process that teaches proper camera movement and visual feedback monitoring.

**4. Start Calibration**

Click the **Start** button to begin. The tool will guide you through positioning steps.

**5. Calibrate the Camera**

- **Circle Alignment**: Move and rotate the camera so the blue circle overlaps perfectly with each red circle. Match size, form, and position as closely as possible. The blue circle acts as a dynamic light spot, changing based on distance and angle.
- **On-Screen Instructions**: Pay attention to guidance text appearing at the top. Receive real-time feedback on camera positioning.
- **Pan and Tilt Indicators**: Side and bottom indicators assess pan/tilt alignment quality. Keep the indicator point in the central area for correct orientation. "GOOD" alignment keeps points centered; "BAD" alignment shows points outside the center area.

**6. Finalize Calibration**

Click the **Exit** button to save new parameters by overwriting the factory `SNxxxxxxx.conf` file. You can reset to factory settings using ZED Explorer if needed.

### Post-Calibration Verification

Use the **ZED Depth Viewer** tool to evaluate the quality of images, depth maps, and point clouds produced after calibration adjustments.
